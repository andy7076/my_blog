# 纯浏览器端AI抠图工具：图片不离开你的电脑

![AI抠图工具界面](https://static.yyfollower.com/removebg_screenshot.png)

做设计、写博客、做PPT的时候，抠图是个高频需求。但市面上的在线抠图工具，要么收费，要么要把图片上传到他们的服务器——隐私？不存在的。

于是我自己搞了一个：**完全在浏览器本地运行**的AI抠图工具，图片从头到尾不离开你的电脑。

## ✨ 核心特性

### 🏠 纯本地运行，不上传任何数据
这是最大的卖点。AI模型（RMBG-1.4）下载到浏览器本地缓存（IndexedDB），推理在 Web Worker + WebGPU 里跑，你的图片数据从头到尾没离开过本地。关掉网页，啥痕迹都没有。

### ⚡ WebGPU 加速
用 Chrome 的 WebGPU API 做推理加速，支持 GPU 的机器上速度飞快。模型加载完成后，抠一张图基本秒出预览。

### 🖌️ 编辑工具
AI 抠完不是终点——你还可以手动修：
- **画笔添加/擦除**：AI 没抠干净的地方手动补一刀
- **撤销/重做**：改错了随时回退
- **缩放/平移**：精细调整时放大看细节
- **预览背景切换**：棋盘格、纯色背景随意换，看抠图效果一目了然

### 📦 支持多种格式
上传 PNG、JPG、WebP 都行，导出透明背景 PNG，直接下载。

### 🌍 中英文双语
自动跟随浏览器语言切换，不用额外设置。

## 🛠️ 技术栈

- **框架**：Next.js（App Router）
- **AI 模型**：RMBG-1.4（浏览器端通过 WebGPU 推理）
- **推理引擎**：Web Worker + WebGPU
- **存储**：IndexedDB（模型缓存）
- **部署**：Nginx 反向代理 + systemd
